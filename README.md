# Online Food Delivery System

A Java Spring Boot application for managing food orders, restaurants, and deliveries. This project implements full CRUD (Create, Read, Update, Delete) operations using a file-based storage system.

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/it25100172/OnlineFoodDeliverySystem.git
   ```
2. **Run the application**:
   ```bash
   ./mvnw spring-boot:run
   ```
3. **Access the application**:
   Open [http://localhost:8081](http://localhost:8081) in your browser.

## 📂 Project Structure
- `src/main/java`: Backend logic (Controllers, Services, Models)
- `src/main/resources/DB`: Text-based data storage
- `src/main/resources/static`: Frontend HTML/CSS files
